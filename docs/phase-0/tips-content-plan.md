## C. 시드 글 15편

### 1. 처음 등반장에 가기 전 30분 — 입문자가 알아야 할 7가지

- **요약**: 첫 방문 전 알아야 할 핵심 7가지(예약·복장·신발 대여·매너·기본 안전). 한국 지점들의 일반적 운영 방식과 첫 방문자가 흔히 놓치는 포인트. 막연한 두려움을 구체적 체크리스트로 교체.

### 2. 정확한 풋워크 — 발끝 vs 인사이드 엣지 사용 시점

- **숙련도**: 입문 → 초급 / **카테고리**: 기본기
- **요약**: 풋워크는 클라이밍의 절반. 발끝(toe)과 인사이드 엣지의 사용 시점, 시각화하며 발 놓기, 흔한 실수(발 보지 않고 놓기). 정확한 풋워크는 손목과 손가락 부담을 크게 줄임.
- **출처**:
  - trainingforclimbing.com — Eric Hörst, "Footwork Fundamentals" [URL: 큐레이터가 발행 전 확정]
  - YouTube Movement For Climbers — "Footwork Drills for Beginners" [URL: 큐레이터가 발행 전 확정]
  - climbing.com — "Five Footwork Drills Every Climber Should Master" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 한국 볼더링 짐의 컬러 홀드 시스템에서 풋홀드 색이 별도 지정되지 않은 경우가 많음 — "모든 홀드 사용" vs "지정 풋홀드" 룰 차이 안내

### 3. 처음 사는 클라이밍 슈즈 — 사이즈와 종류 가이드

- **숙련도**: 입문 / **카테고리**: 장비
- **요약**: 입문자가 흔히 빠지는 함정(너무 작은 사이즈, 공격적 다운턴 슈즈 선택). 평평한(flat) 슈즈로 시작하는 이유, 양말 vs 맨발, 한국 발 모양에 맞는 라스트(폭 넓은 모델) 추천 카테고리.
- **출처**:

  - YouTube Hooper's Beta — "Climbing Shoes Explained" [URL: 큐레이터가 발행 전 확정]
  - ukclimbing.com — UKC Gear Review, "Beginner Climbing Shoe Guide" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 한국 주요 슈즈 브랜드 구매처(스포츠클라이밍·라셀라·국내 단독수입사), AS·재창(resole) 가능 업체

### 4. 한국 볼더링 색깔 등급 이해하기 — 컬러 홀드 시스템 입문

- **숙련도**: 입문 / **카테고리**: 한국 환경
- **요약**: 한국 볼더링 짐의 컬러 시스템(흰→노→주→초→파→빨→보→검 등 지점별 차이). V스케일·폰트 스케일과의 대응. "지점마다 다르다"는 사실 + 환산 어려움 + 단계 표기 룰.
- **출처**:

  - climbing.com — "Climbing Grade Systems Explained" [URL: 큐레이터가 발행 전 확정]
  - ukclimbing.com — "Bouldering Grades: V-Scale vs Font Scale" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 더클라임·어센드·락트리 등 주요 체인의 컬러 체계 비교표. "지점별 등급의 절대 비교는 무리"라는 점 강조.

### 5. 흔한 오해 — "행보드는 초보자에게 절대 금지"는 사실일까

- **숙련도**: 입문~초급 / **카테고리**: 흔한 오해 + 트레이닝
- **요약**: 인터넷에서 흔히 보이는 "행보드는 손가락 다친다, 초보자 금지" 통념. 최근 연구·코치 가이드는 입문자도 조심스럽게 시작 가능하다는 입장. 단, 강도·자세·세트 관리 절차 필수. 진짜 위험은 행보드 자체가 아니라 잘못된 적용.
- **출처**:
  - trainingforclimbing.com — Eric Hörst, "Hangboard Training for Beginners?" [URL: 큐레이터가 발행 전 확정]
  - latticetraining.com — Tom Randall, "Should Beginners Hangboard?" [URL: 큐레이터가 발행 전 확정]
  - PubMed — Schöffl et al., "Finger flexor pulley injuries in rock climbers" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 한국 짐에 행보드 비치 일반화, 자격 코치 없이 시작하는 사용자 다수 — 무리하지 않는 시작 프로토콜

### 6. 데드행, 안전한 시작 방법 — 부상 없이 손가락 키우기

- **숙련도**: 초급 / **카테고리**: 트레이닝 루틴 + 부상 예방
- **요약**: 데드행(맥스 행) 프로토콜의 기본. 워밍업 → 짧은 세트 → 점진적 증량. 손가락 그립 타입(crimp vs open hand), 절대 하지 말 것(통증 무시, 매일 강도 행).
- **출처**:
  - trainingforclimbing.com — Eric Hörst, "Hangboard Protocol: Repeaters vs Max Hangs" [URL: 큐레이터가 발행 전 확정]
  - latticetraining.com — Ollie Torr, "Beginner Hangboard Programme" [URL: 큐레이터가 발행 전 확정]
  - PubMed — López-Rivera & González-Badillo, "Finger flexor strength training in rock climbers" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 한국 짐 행보드 일반 비치 모델(Beastmaker, Tension Block, Lattice Edge 등), 가정 설치 시 도어 마운트 안전 주의

### 7. 손가락 풀리(A2) 부상 — 예방·증상·복귀 단계

- **숙련도**: 초급~상급 (전체) / **카테고리**: 부상 예방·재활
- **요약**: A2 풀리는 클라이밍 손가락 부상 1위. 부상 메커니즘, 통증 부위, 즉시 중단 기준, 회복 단계(2~12주). 무리한 재발 방지.
- **출처**:

  - theclimbingdoctor.com — Dr. Jared Vagy, "Pulley Injury Rehab Protocol" [URL: 큐레이터가 발행 전 확정]
  - YouTube Hooper's Beta — "A2 Pulley Injury: Identify and Rehabilitate" [URL: 큐레이터가 발행 전 확정]
- **한국 현지화 메모**: 국내 클라이머 친화 정형외과·재활의학과 명단(서울/경기) — 검증 후. 의료 면책 문구 필수.

### 8. 어깨 충돌증후군 예방 — 등반 전후 7분 루틴

- **숙련도**: 초급 / **카테고리**: 부상 예방·재활
- **요약**: 클라이밍은 어깨 사용 비중이 큼. 회전근개·견갑골 안정화 운동. 등반 전 워밍업 5가지 + 등반 후 쿨다운 2가지. 만성화 전 조기 발견 신호.

- **한국 현지화 메모**: 한국 사용자가 자주 호소하는 어깨 통증 패턴 (스마트폰·책상 자세 + 클라이밍 결합)

### 9. 리드 공포 극복 — 추락 연습 단계별 프로토콜

- **숙련도**: 초급~중급 / **카테고리**: 멘탈 + 무브먼트
- **요약**: 한국 사용자가 가장 어려워하는 단계 — 리드 추락. 1단계(낮은 위치 컨트롤 추락) → 5단계(볼트 위 추락)까지 점진적 노출. 빌레이어와의 신뢰, 안전 추락 자세.

### 10. 다이노 입문 — 떨어져도 안전한 첫 다이노

- **요약**: 다이노 메커닉(다리 추진 → 손 타이밍), 안전한 매트 위치, 첫 시도 시 자주 하는 실수. 동영상 분석 활용 팁.

### 11. 4주 트레이닝 사이클 — 정체기 돌파를 위한 단순 프로그램

- **숙련도**: 중급 / **카테고리**: 트레이닝 루틴
- **요약**: 같은 V급 6개월 정체 시 시도할 4주 사이클(주 1: 강도, 주 2: 볼륨, 주 3: 강도, 주 4: 회복). 행보드·캠퍼스·실등반 분배. 진척 측정 방법.

### 12. 실내→자연암 첫 전환 — 무엇이 다른가

- **숙련도**: 중급 / **카테고리**: 야외 전환

### 13. 캠퍼스보드 — 시작하기 전 알아야 할 모든 것

## D. 사실 확인 절차

### D.1 발행 워크플로우 (4단계)

```
1) 시커 초안 작성
   ├ 다출처 최소 2개로 사실 교차 확인
   ├ 자기 언어로 재구성 (표절 검사)
   └ 한국 현지화 메모 작성

2) 다출처 확인 (시커 self-review)
   ├ 의료/재활/영양 글: 학술 출처 최소 1개 필수
   ├ 모든 URL을 라이브 클릭해 접속 가능 확인
   └ 인용 텍스트 정확성

3) 패러독스 리뷰 (필수 게이트)
   ├ 사실 정확성
   ├ 출처 적절성 (신뢰도)
   ├ 표절 여부 (자기 언어 재구성 정도)
   └ 면책 문구 누락 여부

4) 발행 + 사후 모니터링
   ├ 댓글에서 사실 오류 지적 시 처리 절차 (§D.4)
   └ 분기 1회 정기 재검토 (출처 변경·새 연구)
```

### D.3 표절 방지 기준

| 허용 | 금지 |
|---|---|
| 출처를 명시한 짧은 직접 인용 (1~2문장, 따옴표) | 단락 단위 복붙 |
| 핵심 개념을 자기 언어로 재구성 | 문장 구조 그대로 한국어 번역 |
| 출처의 데이터(표·수치)를 인용 + 출처 명시 | 출처 없는 데이터 사용 |
| 출처를 종합해 본인 결론 도출 | 한 출처에만 의존한 글 |

### D.4 사용자 사실 오류 지적 처리 (5단계)

1. 모더레이터가 댓글 확인 (24시간 이내 1차)
2. 출처 재검토 (시커)
3. 사실이면 → 본문 수정 + 수정 로그(revision)에 기록 + 댓글에 답변
4. 사실 아니면 → 댓글에 친절한 반박 + 추가 출처 제시
5. 판단 어려우면 → 외부 전문가 자문 (해당 분야 코치·의사)
